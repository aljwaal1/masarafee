# مصروفي الذكي

تطبيق Flutter عربي خفيف لإدارة الدخل والمصاريف والالتزامات الشهرية بدون إنترنت.

## النسخة الأولى

- إضافة دخل.
- إضافة مصروف.
- إضافة التزام قادم مثل إيجار أو فاتورة أو قسط.
- ملخص شهري: الدخل، المصروف، المتبقي، والالتزامات غير المدفوعة.
- فلترة حسب الشهر والنوع والتصنيف.
- حفظ البيانات محليًا على الجهاز.
- واجهة عربية هادئة ومناسبة للهاتف.

## طريقة البناء عبر GitHub

ارفع الملفات إلى مستودع GitHub، ثم شغل workflow باسم:

`Build APK Direct`

بعد انتهاء البناء سيظهر ملف التطبيق مباشرة داخل:

`apk/masroofi-smart.apk`

هذا الملف APK مباشر، وليس Release، وليس ZIP.

## ملاحظات

- التطبيق Android فقط.
- لا يحتاج إنترنت أثناء الاستخدام.
- لا يحتاج صلاحيات حساسة.
- الحد الأدنى المقترح: Android 6 وما فوق.
